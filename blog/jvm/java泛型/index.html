Java选择的泛型实现方式叫作“类型擦除式泛型”(Type Erasure Generics)

Java 语言中的泛型，它只在程序源码中存在，在编译后的字节码文件中，就已经替换为原来的原生类型(Raw Type，也称为裸类 型)了，并且在相应的地方插入了强制转型代码，因此，对于运行期的 Java 语言来说，ArrayList与 ArrayList就是同一 个类，所以泛型技术实际上是 Java 语言的一颗语法糖，Java 语言中的泛型实现方法称为类型擦除，基于这种方法实现的泛型称为伪泛 型。
