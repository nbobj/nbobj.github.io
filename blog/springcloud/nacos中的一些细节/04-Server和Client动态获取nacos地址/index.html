客户端动态获取Nacos地址

这样配置后客户端就能动态的获取Nacos的地址了。

server-addr不能配置，不然会使用server-addr的值

需要提供一个HTTP服务

源码：

首先会通过NacosDiscoveryProperties的getNacosProperties方法

处理endpoint，然后会在NacosNamingService初始化的时候创建了一个NamingProxy对象。在NamingProxy对象初始化时会调用其initRefreshTask方法，在这个方法中会添加两个定时任务：

  • 一个是30s运行一次的获取nacos地址任务

  • 一个是5s运行一次的登录任

    这个任务就是nacos需要登录的话，就会定时执行

    nacos地址可能会变
